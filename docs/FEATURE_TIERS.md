## Tier Selection Guide

### Choose **Free** If You:
- Want to try HookRing risk-free
- Create 1вҖ“3 ringtones per month
- Only need basic MP3 exports
- Don't mind ads

### Choose **One-Off** If You:
- Want a single extra export this month
- Don't want a recurring subscription
- Like flexibility over commitment
- Want to test premium features once

### Choose **Premium Monthly** If You:
- Create multiple ringtones monthly
- Want flexibility to cancel anytime
- Want premium audio quality
- Need reliable customer support

### Choose **Premium Annual** If You:
- Plan to use HookRing long-term
- Create ringtones regularly
- Want 33% savings
- Prefer predictable yearly costs

## Tier Upgrades & Downgrades

### Upgrade Flow
1. User clicks "Upgrade" in app
2. Select desired plan (Monthly or Annual)
3. Enter payment details (Stripe/Apple Pay)
4. Instant upgrade with immediate access
5. Confirmation email sent

### Downgrade Flow
1. User navigates to Settings вҶ’ Subscription
2. Click "Manage Subscription"
3. Select "Cancel Subscription"
4. Immediate downgrade to Free tier
5. Confirmation email sent
6. Existing exports accessible for 30 days

### Refund Policy
- **Annual Plans:** 30-day money-back guarantee
- **Monthly Plans:** Cancellation takes effect at next billing date
- **One-Off Purchases:** No refunds (instant delivery)
